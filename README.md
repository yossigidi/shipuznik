# שיפוצניק

אפליקציית מובייל-פירסט (PWA) לשיפוצניקים ישראלים — ניהול הצעות מחיר, מחשבון כמויות חומרים, מחשבון עלויות עם רווח, חתימה דיגיטלית של הלקוח.

## מה כלול ב-MVP

- **הצעת מחיר ויזואלית** — בחירת פריטי עבודה מקטלוג (ריצוף, צבע, אינסטלציה, חשמל, גבס, דלתות), הגדרת כמות+מחיר, סיכום
- **מחשבון כמויות חומרים** — סוג חדר + מידות → חישוב קרמיקה, דבק, רובה, צבע, פנלים (כולל עודף לפסולת)
- **מחשבון עלויות + רווח** — עלות חומרים בקנייה + תוספת רווח (קרמיקה 25%, אחרים 10%) → מחיר ללקוח
- **תצוגת לקוח + חתימה דיגיטלית** — מסך שהלקוח מקבל בלינק, אישור עם חתימה ב-Canvas
- **שיתוף בוואטסאפ** — כל הצעה/רשימה ניתנת לשליחה מיידית

## טכנולוגיות

- React 18 + Vite 8
- Tailwind CSS 3
- Lucide React (אייקונים)
- אחסון: localStorage (בעתיד יוחלף ב-Firebase Firestore)
- ניווט: state-based, ללא react-router

## פיתוח

```bash
npm install
npm run dev      # פיתוח מקומי
npm run build    # build לפרודקשן (פלט ב-dist/)
npm run preview  # תצוגת build מקומי
```

## דיפלוי

מאוחסן ב-Cloudflare Pages עם push אוטומטי מ-main.

- **Build command:** `npm run build`
- **Build output directory:** `dist`
- **Node version:** 20+
